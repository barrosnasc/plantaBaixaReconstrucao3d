\begin{resumo}
\ABNTEXchapterfont
Este trabalho propõe um sistema para reconstrução de modelos 3D a partir de plantas baixas, com foco na região de Teresina. A metodologia envolve a revisão das técnicas presentes na literatura, a seleção de uma arquitetura compatível com o hardware disponível, o treinamento do modelo com datasets consolidados e a validação com plantas digitalizadas. O sistema realiza a detecção de elementos arquitetônicos e gera um modelo 3D correspondente, exibido em uma interface web para comparação entre a planta original e a planta processada.

\textbf{Palavras-chaves}: Processamento de Imagem; Modelo 3D; Reconstrução 3D; Planta baixa.
\end{resumo}
%--------------------------------------------------x------------------------------------------------------


% Abstract
%---------------------------------------------------------------------------------------------------------
\begin{resumo}[Abstract]
\ABNTEXchapterfont
This project proposes a system for reconstructing 3D models from floor plans, with a focus on floor plan data from the region of Teresina. The methodology includes reviewing techniques presented in the literature, selecting an architecture compatible with the available hardware, training the model using consolidated datasets, and validating it with digitized floor plans. The system performs the detection of architectural elements and generates a corresponding 3D model, which is displayed in a web interface for comparison between the original floor plan and the processed one.

\end{resumo}
